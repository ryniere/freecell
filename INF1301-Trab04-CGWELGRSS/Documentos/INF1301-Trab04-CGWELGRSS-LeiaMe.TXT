Apresentaçăo

Este texto descreve o conteúdo do arquivo INF1301-Trab02-CGEGRS.zip
 que contém a documentaçăo e os módulos que compoem o jogo FREECELL.

 Instalaçăo

Há duas maneiras de executar o teste dos módulos: o que recompila os módulos e o que apenas executa.
Para executar em conjunto com a compilaçăo, basta abrir uma janela de prompt de comando do windows XP mais recente, ir para o diretório
INF1301-Trab02-CGWELGRSS\BATCHES, e executar o arquivo Tudo.bat, se der algum erro de compilaçăo, pegue o arquivo vsvars32.bat na pastas commons
do visual studio instalado no seu pc e substitua na pasta BATCHES.
A segunda forma de execuçăo é executar o arquivo TestaTudo.bat no mesmo diretório.

Arquivos de código fonte(na pasta FONTES)


Módulo Carta

CARTA.H – Arquivo header do módulo carta. Ele contém a interface menos restrita do módulo, disponibilizando um ponteiro para o tipo carta e funçőes de 
	consulta aos dados da estrutura.
	
PRIVILEGIOCARTA.H – Arquivo header do módulo carta. Ele contém a interface mais restrita do módulo, disponibilizando uma funçăo de criaçăo e destruiçăo 
	de estruturas carta.

CARTA.C – Arquivo que contém a implementaçăo das funçőes de interface do módulo carta, além de funçőes auxiliares.

TST_ESPEC.H – Arquivo header do módulo de teste referente ao módulo carta. Este arquivo possui a interface do módulo de teste específico 
que é acoplado ao módulo de teste genérico a fim de automatizar os testes sobre o módulo.

TESTCARTA.C – Arquivo que contém a implementaçăo da interface de teste específico do módulo carta.

Utilitários e Ferramentas

GMAKE.EXE – é um programa utilitário que gera o arquivo de diretivas MAKE a partir
de uma descriçăo de composiçăo de um construto.

GMake.parm – é o arquivo de diretivas para o GMake utilizado para gerar diretivas
MAKE para o projeto arcabouço de teste utilizando a plataforma MS-Visual C/C++. Este
arquivo deve ser ajustado para poder compilar visando outras plataformas e outros
projetos.

GeraMake.bat – é um batch utilizado para gerar as diretivas MAKE de um determinado
construto exemplo.

Compila.bat – compila um programa utilizando um arquivo de diretivas de MAKE
geradas por GMAKE. Este batch assume que o ambiente (environment, set, path) esteja
corretamente configurado para que o compilador a ser utilizado compile corretamente.

GeraT2 - Gera os .make de todos os modulos do trabalho 2.

GeraT3 - Gera os .make de todos os modulos do trabalho 3.

GeraT4 - Gera o .make do modulo lista.

CompilaT2.bat – compila todos os construtos utilizados no trabalho 2 e gera a biblioteca FreeCell.lib com eles.

CompilaT3.bat – compila todos os construtos utilizados no trabalho 3 usando a biblioteca FreeCell.lib.

CompilaT4.bat – compila o modulo lista.

LimpaTudo.bat – elimina todos os arquivos que săo gerados por intermédio de alguma
ferramenta.

EXBESTATC.C, EXBESTAT.EXE – é um pequeno programa que permite exibir o arquivo
de acumulaçăo das estatísticas de teste. Quando um teste for formado por diversas
massas de teste (scripts de teste) torna-se conveniente acumular as estatísticas relativas
a cada massa, de modo que, ao final do teste de todas elas, possa-se exibir o resultado
acumulado destas estatísticas. O arquivo TestaTudo.bat ilustra o uso desse programa.

CompileBanner.C, CompileBanner.exe – é um pequeno programa que exibe um
separador no vídeo. O programa é utilizado para exibir separadores ao compilar construtos,
facilitando a compreensăo de uma compilaçăo agregada de diversos construtos.

TestaTudo.bat – Arquivo que auxilia na execuçăo dos testes automatizados de todos os módulos. Este arquivo contém comandos do prompt do Windows que executa o arquivo TRAB2.exe passando como parâmetro o arquivo de script.

Tudo.bat - Arquivo que coordena a compilaçăo e execuçăo dos testes automatizados.

Jogar.bat - Arquivo que executa o jogo FreeCell.
